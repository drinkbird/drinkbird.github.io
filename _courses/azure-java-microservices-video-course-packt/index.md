*Microservices Development on Azure with Java* was produced in collaboration with Packt Publishing and released in October 2018. Although the code examples use Java, the demonstrated patterns translate directly to other platforms such as .NET Core.

<p><iframe width="560" height="315" src="https://www.youtube-nocookie.com/embed/aB3OzV0HlIM" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe></p>

The course covers building, deploying, and operating containerized Java microservices in the cloud using Docker and the Azure Kubernetes Service, while leveraging building blocks of an open and flexible cloud platform with excellent Java support. It includes a number of patterns that fit naturally into the microservices architectural style, as well as DevOps practices to manage the full application lifecycle from laptop to cloud.

Technologies covered include Azure DevOps, Docker, Kubernetes, SendGrid, Azure Cosmos DB, Redis, and more.
